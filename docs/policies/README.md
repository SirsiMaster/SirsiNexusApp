## üîí Key Security Requirements

### Multi-Factor Authentication (MFA)

**MFA is MANDATORY for:**
- All administrator and employee accounts
- All consumer/end-user accounts
- Access to financial integrations (Plaid, Stripe)
- Document signing operations
- Sensitive data access

### Data Protection

**AES-256 encryption** for all data at rest  
**TLS 1.3** for all data in transit  
**Google Cloud KMS** for key management
