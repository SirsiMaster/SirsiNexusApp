## üé® UNIVERSAL DESIGN PRINCIPLES

### Typography Hierarchy:
- **H1 (Slide titles)**: 3-3.5rem, weight 800, letter-spacing -0.03em
- **Numbers**: 4-5rem, weight 900, gradient fill
- **Body**: 1rem, weight 400-500, line-height 1.6
- **Labels**: 0.8-0.9rem, weight 600, letter-spacing 0.05em

### Color Usage:
- **Primary Green**: Key numbers, CTAs, success indicators
- **Dark**: #0F172A for text (not pure black)
- **Gray Scale**: 3 levels only (text, secondary, muted)
- **White**: Clean backgrounds, cards
- **Accents**: ONLY green spectrum (light to dark)

### Iconography:
- Custom line icons (not emoji in final)
- 24-32px size range
- 2px stroke weight
- Consistent style throughout
- Green accent color on icons

### Shadows & Depth:
- **Cards**: Soft, subtle shadows (not heavy drop shadows)
- **Hover states**: Lift 4-8px with green-tinted shadow
- **Important elements**: Add subtle green glow
- **Layering**: Use subtle gradients for depth

### Spacing:
- **More white space**: Don't cram
- **Breathing room**: 30-40px between major elements
- **Consistent gaps**: 20px, 30px, 40px (not random)
- **Edge margins**: 60px minimum from screen edge

### Motion & Polish:
- **Hover effects**: Subtle lift + shadow
- **Number counters**: Count up on slide enter
- **Fade ins**: Stagger for bullet points
- **Progress indicator**: Show slide position
- **Smooth transitions**: 0.35s cubic-bezier
